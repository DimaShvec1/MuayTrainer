Muay Thai Trainer

Muay Thai Trainer — это мобильное Android-приложение для изучения техники тайского бокса (Муай Тай) и организации тренировочного процесса. Проект разработан с использованием актуальных компонентов Android Jetpack и демонстрирует реализацию архитектурного паттерна MVVM, работу с мультимедиа и интерактивными элементами UI.

Технический стек

Язык разработки: Kotlin
Минимальная версия SDK: 24 (Android 7.0 Nougat)
Целевая версия SDK: 34 (Android 14)
Система сборки: Gradle (Kotlin DSL)
Архитектурный паттерн: MVVM (Model-View-ViewModel)
Компоненты UI: Material Design 3, ViewBinding, RecyclerView, BottomNavigationView
Работа с видео: YouTube Player API
Мультимедиа: MediaPlayer (звуковые эффекты), Vibrator (тактильная обратная связь)
Навигация: Jetpack Navigation Component
Анимации: кастомные анимации (fade_in, slide_up, scale_in)
Работа с изображениями: Vector Asset, Drawable ресурсы

Детальный функционал

1. Библиотека техник (ui.techniques)

Отображение списка из 12 базовых ударов Муай Тай.
Категоризация ударов по типам: удары руками и удары ногами.
Детальный просмотр техники с видео-примером через YouTube Player API.
Анимированные карточки с эффектом появления при скролле (RecyclerView + кастомная анимация).
Реактивное обновление интерфейса.

2. Боксёрский таймер (ui.timer)

Настройка времени раунда через ручной ввод минут и секунд.
Автоматическое переключение между раундами и отдыхом (3 раунда по 1 минуте отдыха).
Визуальный прогресс-бар в виде круга (ImageView с изменяемым уровнем).
Звуковое сопровождение: сигналы начала и конца раунда.
Тактильная обратная связь: вибрация в конце раунда.
Фоновый режим работы: таймер продолжает отсчёт при переключении между вкладками навигации.
Управление: кнопки Старт, Пауза, Сброс.

3. ОФП тренировка (ui.workout)

8 упражнений для общей физической подготовки.
Интервальный режим: 45 секунд работы, 15 секунд отдыха.
Отображение текущего упражнения и статуса тренировки.
Звуковые сигналы для смены упражнений.
Отслеживание прогресса выполнения.

4. Словарь терминов (ui.dictionary)

Более 30 терминов Муай Тай с переводом и пояснениями.
Категоризация по разделам: удары, защита, традиции.
Удобная навигация по разделам.

5. Приветственный экран и навигация (ui.welcome, MainActivity)

Экран приветствия с анимированным переходом.
Нижняя навигация с переключением между четырьмя основными экранами.
Плавные переходы между фрагментами с кастомными анимациями.

Дизайн и UX

Тёмная тема с градиентным фоном (красный -> чёрный).
Неоновые акценты в стиле боевых искусств.
Стеклянные элементы управления (полупрозрачные кнопки с обводкой).
Плавные анимации переходов между экранами.
Адаптивный интерфейс под разные размеры экранов.
Кастомизированные карточки для отображения техник.

Сборка и запуск

Убедитесь, что у вас установлена актуальная версия Android Studio и Android SDK 34.

Скачайте данный репозиторий:
git clone https://github.com/DimaShvec1/MuayTrainer.git

Откройте проект в Android Studio и дождитесь завершения индексации и синхронизации Gradle-скриптов.

Выполните сборку проекта через меню Build -> Make Project или запустите приложение напрямую на подключенном устройстве/эмуляторе с помощью кнопки Run.
